## 加入度（ADD）是什麼？

拿到驗光處方單，很多人會盯著其中一欄寫著 **ADD**、後面接個像 `+2.00` 的數字，一頭霧水。這個 ADD 就是**加入度（Addition）**，講白話就是：

> **為了「看近」，在你原本的「看遠」度數之外，額外加上去的一段正度數（凸透鏡度數）。**

單位是**屈光度（D）**，前面通常掛個「＋」號。它不是一個自己獨立的度數，而是**疊在遠用度數上面的一段補償**。它會叫「加入」度，原因就在這。

---

## 為什麼看近需要「加」度數？先懂老花

要搞懂 ADD，得先認識**老花（presbyopia）**這個生理現象。

年輕的時候，眼睛看近會靠**睫狀肌收縮、水晶體變厚**來增加屈光力，這個自動對焦的本事叫**調節力**。年紀慢慢上來，水晶體越來越硬、調節力自然往下掉，看近就變得吃力。**這是每個人都會走到的生理老化，不是病。**

等眼睛自己「調」出來的度數不夠用了，就得靠鏡片幫忙補上這段差額。**補上去的這段正度數，就是加入度 ADD。**調節力剩得越少，要補的 ADD 就越多，ADD 會跟年齡有關，道理就在這。

## 老花度數怎麼看？近用度數這樣算

處方單上通常會把**遠用度數**跟 **ADD** 分開列，而你真正看近時用到的度數，是這兩個加起來：

> **近用度數 ＝ 遠用球面度數（Sph）＋ 加入度（ADD）**

舉幾個例子（僅示範換算邏輯，實際以驗光結果為準）：

| 遠用度數（Sph） | 加入度（ADD） | 換算後的近用度數 |
|----------------|--------------|-----------------|
| 0.00（遠方本來就清楚） | +2.00 | +2.00 |
| −2.00（原本近視） | +2.00 | 0.00 |
| −4.00（較深近視） | +2.50 | −1.50 |
| +1.50（原本遠視） | +2.00 | +3.50 |

表格裡藏了一個有意思的點：**本來就有近視的人**，看近因為加了正度數，近用度數反而會往「0」靠、甚至變成正的。這就解釋了為什麼有些近視的人上了年紀，看近時把眼鏡拿下來反而覺得清楚。

## ADD 的幾個常見特性

| 問題 | 一般情況 |
|------|---------|
| **左右眼 ADD 會一樣嗎？** | 通常左右眼相同 |
| **ADD 會隨年齡變嗎？** | 大致隨年齡增加，因為調節力持續下降 |
| **ADD 有沒有上限？** | 到一定年齡後多半趨於穩定，因為調節力已所剩不多 |
| **ADD 越高越好嗎？** | 不是，過高會把清楚的近點推得太近 |

要提醒一下，上面講的都是**一般趨勢跟換算邏輯**。實際上每個人的 ADD 要多少，牽涉到用眼距離、平常的用眼習慣跟雙眼協調，是得**由驗光人員實際量測**才能定的參數，這篇只負責把名詞跟算法講清楚。

## 常見誤解 / FAQ

### Q1：ADD 兩眼會不一樣嗎？

一般情況下，**左右眼的加入度是相同的**。因為 ADD 反映的是「調節力隨年齡下降」的程度，兩眼年齡一樣，要補的近用度數通常也一致。至於兩眼的**遠用度數**常常不一樣，那是另一回事了。

### Q2：ADD 是不是越高越好？

不是。ADD 越高，代表鏡片幫你補的正度數越多，**看得最清楚的那個「近點」會被推得越近**，太高反而會讓舒服的閱讀距離變得不自然。合適的 ADD 是「剛好補足看近需要的」，不是越高越好，這得看你實際的用眼距離來定。

### Q3：為什麼 ADD 會隨年齡增加？

因為 ADD 補的就是「調節力不夠」的那段差額。年紀越大，水晶體越硬、調節力越低，眼睛自己能調出來的度數越少，要靠鏡片補的部分就越多，ADD 自然往上走。到了一定年齡，調節力所剩無幾，ADD 大多也就趨於穩定了。

### Q4：處方單上的 ADD 怎麼換算成近用度數？

公式是：**近用度數 ＝ 遠用球面度數 ＋ ADD**。比方遠用是 −2.00、ADD 是 +2.00，近用度數就是 0.00；遠用 0.00、ADD +2.50，近用度數就是 +2.50。散光（Cyl）跟軸度（Axis）通常沿用遠用的數值，不會跟著 ADD 改變。

## 小結

**加入度（ADD）就是為了看近，在遠用度數之外多加上去的一段正度數**，用來補老花之後不夠用的調節力；近用度數就是「遠用度數＋ADD」。這個名詞看懂了，處方單上那串數字也就不再像天書了。
