## 蔡司 Individual² 和一般鏡片，差在哪裡？

### 一般鏡片的製造邏輯

一般非訂製鏡片（包括很多知名品牌的標準系列）採用**庫存半成品**製造：工廠預先生產大量不同度數組合的半成品，配鏡時依你的度數切割成型。

問題在於：鏡片的光學設計，是基於**統計平均值**建立的。平均臉型、平均鼻高、平均眼睛位置。你的臉不是平均值。

### 蔡司 Individual² 的訂製邏輯

Individual² 的計算需要輸入以下個人化數據：

| 參數 | 說明 |
|------|------|
| 瞳距（單眼分開測量） | 左右眼各自到鼻梁的距離 |
| 瞳高 | 瞳孔在鏡框中的垂直位置 |
| **頂點距離（Vertex Distance）** | 角膜到鏡片後表面的距離，影響有效度數 |
| **前傾角（Pantoscopic Tilt）** | 鏡框向前傾斜的角度，影響散光軸向補正 |
| **環曲角（Face Form Angle）** | 鏡框弧度包覆臉部的角度 |
| 閱讀距離 | 慣用近距離工作距離 |

這些數據輸入蔡司的 i.Profiler® 系統後，鏡片的每一個光學區域都根據你的眼睛、臉型、配戴習慣重新計算，從源頭製造，不使用庫存半成品。

## 誰最能感受到差異？

**最受益的族群：**

- **高度近視（600 度以上）**：頂點距離補正的誤差在高度數時影響最大，未補正可能造成 25–50 度的有效度數誤差
- **高度散光（200 度以上）**：前傾角未校正會導致散光軸向偏移，Individual² 會補正這個誤差
- **多焦點老花使用者**：訂製的閱讀區域比標準設計更符合你的實際近距離工作需求
- **斜軸散光**：非水平垂直軸向的散光，前傾角補正效果最明顯
- **對視覺品質要求高的專業人士**：長時間用眼、需要精準視覺的職業（設計師、醫師、精密工作者）

**效益較不明顯的族群：**

- 低度近視（300 度以下）、無散光
- 不在乎視覺細膩差異的使用者
- 鏡框選擇平面、小框（前傾角和環曲角影響本來就小）
