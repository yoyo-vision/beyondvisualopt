## 配了多焦點鏡片，戴了一週還是頭暈——正常嗎？

「老闆說適應期要一個月，撐一下就好。」

這句話有一半是對的，有一半是在幫配鏡錯誤找藉口。

漸進多焦點鏡片確實需要適應期，因為你的視覺系統在學習「在不同區域找到對應的焦點」。但**適應期的不舒服是有範圍的**——如果某些症狀超出正常範圍，繼續撐下去只是浪費時間，甚至會讓你更排斥漸進鏡片。

這篇文章拆解漸進鏡片的常見症狀、原因與判斷方式，讓你知道什麼時候該繼續適應，什麼時候該回驗光所。

---

## 先理解漸進鏡片的結構

漸進鏡片不是一片均勻的鏡片。它從上到下分成三個功能區：

```
┌─────────────────────┐
│   遠用區（看遠處）    │ ← 鏡片上方
│                     │
│    過渡走廊（漸變）   │ ← 鏡片中間
│                     │
│   近用區（看近處）    │ ← 鏡片下方
└─────────────────────┘
```

走廊兩側（左右周邊區域）是**像差區**，這裡的影像本來就是扭曲的——這是漸進鏡片的物理限制，無法完全消除，只能透過不同設計縮小影響。

**正常適應期的不舒服：**
- 走廊兩側的輕微扭曲感（看地板時感覺不平）
- 需要主動移動頭部（而不只是眼睛）找到清晰區
- 下樓梯時需要低頭看正前方，而不是往下看

**不在正常範圍內的症狀：**
- 頭暈、噁心、頭痛持續超過兩週
- 無論怎麼調整視線方向，就是找不到清晰的近用區
- 雙眼看同一目標時有影像高低落差（一隻眼比另一隻高）
- 鏡片上下清晰區域比例明顯不對稱

## 最常見的問題：配鏡高度沒量準

漸進鏡片有一個關鍵參數叫做**配鏡高度（Fitting Height / Seg Height）**——就是鏡片光學中心到鏡框下緣的距離。這個數值決定了「近用區」落在哪裡。

如果配鏡高度量錯了：

| 錯誤情況 | 症狀 |
|---------|------|
| 高度量太高 | 近用區落在太高的位置，看書要非常低頭，不自然 |
| 高度量太低 | 視線往下掃時，近用區在更低的地方，找不到 |
| 左右高度不一致 | 雙眼看近時影像有高低落差，頭暈 |

**配鏡高度必須在你實際配戴鏡架的狀態下量測。** 換句話說，要先挑好框架、調整到正確的佩戴位置，驗光人員在你配戴著鏡架的狀態下，用記號筆標記瞳孔中心，才能量出正確的配鏡高度。

---

## 第二常見的問題：鏡架下滑

配好的時候量得很準，但鼻托或鏡腳沒有調整好，實際戴起來鏡架往下滑 2–3mm——近用區跟著往下跑，你低頭看書時剛好看到的是走廊區而不是近用區，當然一直找不到清楚的焦點。

**解決方式很簡單：** 回驗光所請驗光人員調整鏡架貼合度，確認鏡架服貼、不下滑。不需要重新做鏡片。

---

## 第三個原因：加入度（ADD）設定不對

加入度是遠近之間的度數差。ADD +1.00 代表看近時比看遠多加了 100 度的輔助。

ADD 太高：
- 看近確實清楚，但走廊太短、兩側像差更嚴重
- 更難適應

ADD 太低：
- 看近還是霧的，以為鏡片不行，其實是度數不夠

**加入度需要根據老花實際狀況評估，不是年紀越大 ADD 就越高。** 有些人 55 歲但調節力還不錯，ADD +1.25 就夠；有些人 50 歲但調節力已明顯退化，可能需要 ADD +2.00。

## 什麼情況繼續撐，什麼情況該回診？

| 症狀 | 建議 |
|------|------|
| 周邊輕微扭曲、走路需要注意腳步 | 正常，繼續適應 2–4 週 |
| 看近需要特別低頭才清楚 | 正常，習慣用下視野看近 |
| 頭暈但沒有頭痛，1 週後減輕 | 正常，繼續觀察 |
| 戴了 2 週頭暈沒改善 | 回診確認配鏡高度與鏡架位置 |
| 怎麼找就是找不到清晰的近用區 | 回診，可能是高度或 ADD 問題 |
| 雙眼影像有高低落差 | 儘快回診，可能是配鏡高度左右不一致 |
| 頭痛、噁心、無法正常工作 | 立即回診，停用鏡片 |

## 怎麼加快適應期？

**正確的視線習慣**
- 看遠：視線平視，用鏡片上半部
- 看電腦：視線略低，約在鏡片中段
- 看手機/書：頭微微抬起，眼睛往下看鏡片下方的近用區（不是低頭）

**不要強迫自己「習慣」扭曲感**
如果走廊兩側的扭曲讓你很不舒服，可能是鏡片設計等級不夠高。入門等級漸進鏡片的走廊較窄、像差區較大；高階個人化漸進鏡片（如蔡司 Individual² 或 HOYA iD 系列）的像差分散設計更好，走廊更寬。如果你是初次配漸進鏡片，起點的鏡片選擇確實很重要。

**多戴、不要混用**
適應期間盡量全天配戴漸進鏡片，不要交替使用舊的單光鏡片——視覺系統需要持續的刺激才能建立新習慣。

## 常見問題

**Q：漸進鏡片適應期要多久？**

一般需要 1–4 週。初次配戴者通常 2–3 週，更換新鏡片（度數或設計改變）約 1–2 週。如果超過 4 週還有明顯不適，建議回診評估。

**Q：走路下樓梯時腳步虛是正常的嗎？**

初期是正常的。漸進鏡片的近用區在下方，走路時往下看地板，剛好看到的是近用區或過渡帶，地板影像會稍微模糊或扭曲。適應期間走樓梯時建議把頭微微抬高，用視線的中間偏上方看前方地板。

**Q：我之前配的漸進鏡片完全不能用，是我不適合漸進鏡片嗎？**

不一定。配鏡高度錯誤、鏡架下滑、ADD 設定不對，都可能讓人完全無法適應漸進鏡片。建議重新評估——不是放棄漸進鏡片，而是換一個有完整評估流程的地方重配。
