## 只想配鏡片、框自己帶，眼鏡行會歡迎嗎？

**歡迎，而且這正是我們每天在做的事。**

很多人走進眼鏡行前，會先在門口猶豫一下：「框不是在你們這買的，只想配個鏡片，會不會不太好意思？」這個猶豫我們太懂了。先把話講在前面：在能見度視覺中心，自帶鏡框配鏡片不是什麼通融，就是我們天天在做的服務。網購的新框、收藏很久的精品框、戴習慣的舊框，甚至二手古董框，都歡迎帶來。

那份「不好意思」，多半是一個誤會來的：以為眼鏡行都靠賣框賺錢，你不買框就是白佔人家時間。可是一副眼鏡戴起來舒不舒服、看得清不清楚，**大半是鏡片本身跟量測在決定的**，框說穿了就是把鏡片穩穩掛在你臉上的架子。把鏡片這件事做到好，本來就是配鏡的核心。

## 「只配鏡片」的常見誤會，一次講清楚

| 你可能以為 | 實際情況 |
|---|---|
| 框不是這買的，會被另外加價 | 費用＝鏡片價格＋固定加工費，不因為框是外面買的而加價 |
| 只配鏡片，驗光會隨便量一量 | 驗光與量測跟配整副眼鏡完全同一套標準，一項不少 |
| 自帶框是「小案子」，店家不太想接 | 這是我們的專門服務，網購框、舊框、二手框都收 |
| 什麼框拿來都能直接裝上鏡片 | 需先由驗光人員評估框況，確認可以承作才進行 |

最後一列補一句：評估框況不是要為難你，是先幫你的框把關。鏡框拆裝加工本來就有破損的可能，先看過材質跟結構、確認撐得住再動工，就是不想弄傷你帶來的框，尤其是有價值、有回憶的那種。

## 來店前，帶這兩樣就夠了

自帶框配鏡片其實很單純，來之前準備兩樣東西：

- **你想配的那副鏡框**——網購新框、精品框、舊框、二手古董框都可以。如果框上有明顯鬆脫、裂痕或變形，帶來時先跟我們說一聲。
- **一個「度數參考」**——最方便的是把你現在正在戴的眼鏡一起帶來，我們可以直接量出它的度數當作參考起點。如果手邊有近期的驗光紀錄或度數單，也很好；沒有也沒關係，我們會從頭幫你驗一次。

沒有舊眼鏡、也沒有度數紀錄？完全不影響。完整的驗光流程本來就會重新確認你現在的度數，這一步不會因為你只配鏡片而省略。

## 自帶框配鏡片的來店流程

| 步驟 | 做什麼 |
|---|---|
| 1. 框況評估 | 驗光人員先檢視鏡框材質、結構與老化程度，判斷適不適合拆裝加工 |
| 2. 風險告知 | 若框況有拆裝破損風險，會先誠實說明；你了解並同意後才進行，經評估不適合的會直接建議別勉強 |
| 3. 完整驗光 | 依你的用眼需求做完整驗光流程，確認現在的度數 |
| 4. 配鏡量測 | 瞳距、配鏡高度、頂點距離、鏡架配適逐項量測，把度數對準你的眼睛，而不是只塞進框裡 |
| 5. 選鏡片與報價 | 依需求與預算說明鏡片選項（蔡司 ZEISS、HOYA 簽約經銷），費用＝鏡片價格＋固定加工費，報價清楚 |
| 6. 製作與回店調整 | 完成後現場試戴確認，之後鏡架跑位都歡迎回店調整 |

看到第 3、4 步了嗎？**這才是配鏡片真正花功夫的地方。** 同一個度數，瞳距對不對、鏡片的光學中心有沒有落在你的視線上、鏡框在你臉上的頂點距離多少，看出去的清晰跟舒適差很多。框在哪裡買的，跟這些量測要不要做一點關係都沒有，該做的一項都不會少。

## 配鏡片的功夫在量測跟裝配，框在哪買反而不是重點

如果你手上正拿著一副喜歡的框，卻因為「不是在這買的」不好意思開口，這篇就是想讓你放心：**專門配鏡片，就是我們的本業。** 你負責挑到一副戴得開心的框，把度數配準、鏡片裝好，交給我們就行。

## 常見問題

**Q1：只配鏡片，真的不會比較貴嗎？**
不會因為框是外面買的而加價。費用組成是「鏡片價格＋固定加工費」，鏡片依你選的品牌與等級計價，加工費是固定金額。來店或用 LINE 詢問都能先報價，價格清楚了再決定。

**Q2：我沒有舊眼鏡，也不知道自己度數，可以只配鏡片嗎？**
可以。完整的驗光流程會重新確認你目前的度數，不需要你自己先知道。帶著想配的框來就好；若手邊剛好有在戴的眼鏡，一起帶來能當參考，沒有也不影響。

**Q3：自帶框的驗光會比較隨便嗎？**
不會。自帶框與店內選框的驗光、量測標準完全相同——完整驗光加上瞳距、配鏡高度、頂點距離、鏡架配適。鏡片準不準，量測占一大半，這正是「專門配鏡片」最看重的部分。

**Q4：什麼框都能直接配嗎？**
需要先評估。網購新框、精品框、舊框、二手古董框都歡迎帶來，但會由驗光人員先看過框況與材質，確認可以安全承作才進行；框況有風險的會先告知，經你同意後才做，評估不適合的會誠實建議別勉強。
